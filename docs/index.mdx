# Axone Whitepaper & Documentation

Welcome! Axone is a public PoS layer 1 blockchain built for trust-minimized data sharing. Axone is an open interoperable smart contracts platform for highly customized rules, governance mechanisms and business models to align interests between participants and build useful distributed applications.
